# Base Deployment Guide

## Paketleri Kur

```bash
npm install
```

---

## Compile

```bash
npx hardhat compile
```

---

## Base Sepolia Deploy

```bash
npx hardhat run scripts/deploy.js --network baseSepolia
```

---

## Base Mainnet Deploy

```bash
npx hardhat run scripts/deploy.js --network base
```
